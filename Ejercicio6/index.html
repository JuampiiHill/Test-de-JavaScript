Funciones

  • 1a - ¿Qué es un Array?
  • 1b - ¿Qué es un objeto?
  • 1c - ¿Cuándo es mejor usar objetos o arrays?
  • 1d - ¿Puedo mezclar arrays con objetos o incluso objetos con arrays?

  • 2a- Debemos crear una funcion que pueda recibir cualquier array como parametro e imprima su primer elemento

  • 2b- Crea una función que pueda recibir cualquier array como parámetro e imprima todos sus elementos uno por uno (no se vale imprimir el array completo).

  • 2c- Crea una función que pueda recibir cualquier objeto como parámetro e imprima todos sus elementos uno por uno (no se vale imprimir el objeto completo).

Respuestas

  • 1a - Es una lista de elementos
  • 1b - Es una lista de elementos, pero cada elemento tiene un nombre clave.
  • 1c - Arrays cuando lo que haremos en un elementos es lo mismo en todos los demas (la regla se puede incumplir). Mientras que un objeto cuando los nombres de cada elementos son importantes para nuestro algoritmo
  • 1d - Si, los arrays pueden guardar objetos. y los objetos pueden guardar arrays entre sus propiedades.
